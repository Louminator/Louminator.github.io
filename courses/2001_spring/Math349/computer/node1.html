Overview

Part of learning about linear algebra is learning how to use a computer to help you solve larger problems. In this course, you will acquire some experience using Maple, a computer algebra system. The computer systems used in the classroom are different from the campus ``composer'' (Mahler, Copland, Brahms, Strauss, etc) computer systems because they do not share the same file system. Your username and password is the same for both systems, but if you save a file in class, it will only be stored on the classroom file system.
