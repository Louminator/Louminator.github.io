Maple Problem Set #2

Before doing these problems, create clean new worksheet and put your name at the top. These problems require you to use the plots library. You can load in this library with the command: 

[> with(plots);

Here are some problems to try.

1.
Consider the surface

x2 -y + z2 = 0.

Before using Maple, try to anticipate what you expect it to look like. Use both implicitplot3d and plot3d to graph the surface.

2.
Plot the sphere

x2 + y2 + z2 = 1

using plot3d by patching together two surfaces. Plot the surface using sphereplot.

3.

4.
