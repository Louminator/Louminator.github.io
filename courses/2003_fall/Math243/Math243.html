Special information about Ewing 205 & 207.

Moving files on and off the math server.

When you save files on the workstations in Ewing 205 or 207, they are stored on the math department server, not the UDel composer systems (strauss, copeland, etc).  Thus, you may wish to move files on and off the math server from other machines.  I present two ways to do this.
  1. Using the File Transfer Program (ftp) or secure shell (ssh) from another networked computer like strauss.

    2. FTP is a powerful little program for moving files from one machine to another.  ssh and scp provide the same services with more security.  If you are on strauss for instance, you might type
    ftp newton.math.udel.edu
    Once you enter this command, you will be prompted for your username and password.  Once you have done this, ftp will print some stuff, and you will arrive at a prompt "ftp>".  If you type
    ls
    it will list all the files available in your account on the math server.  You can download a file to strauss by typing
    get <myfile.mws>
    where <myfile.mws> is the file you want.  Similarly, you can move a maple file from strauss to the math server by typing
    put <myfile.mws>
    When you are all done, simple type
    quit
    and you will return to the command line.
  2. Save to a DOS formatted floppy disk.

    3. This may well be the most direct way to do it.  Put your disk into the drive on the workstation.  On your workstation, open an XTERM using the right button menu.  Type into the command line in the XTERM:
    mcopy <myfile.mws> a:
    The file <myfile.mws> refers to the specific filename you chose when you saved your work.  The disk drive should clunk a bit and then you can remove your disk.  If you want to copy a file from your floppy onto the workstation, you would execute a command like
    mcopy a:<myfile.mws> myfile.mws
    Again, the drive should clunk a bit and then the file will be copied on the math server.

Course materials.

Some quick comments on file formats.

When I distribute materials, I will distribute them in an appropriate format.  Generally, I will distribute documents in Adobe's PDF format.  I find that it produces output quality that is good enough to read on a screen or on a printed page, and still the file sizes are modest.  A PDF viewer is freely for most computer architectures and operating systems.  If your machine cannot make use of PDF's, please let me know and I will make other arrangements with you.

Alternative final exam grading policy.


Some students did not master the material for Exam 3 well enough to score well.  Since the final exam in cumulative, I am making the following allowances.  Students may opt for this alternative grading policy as follows.
  • Students must notify me prior to the final via email stating that they choose the Alternative final exam grading policy.
  • Certain problems on the final exam include material covered under Exam 3.  For example, on the Final exam from 2002, questions 5, 6, 8 and 9 involve skills and concepts covered under exam 3.  Note that flux concept in problem 9 is not covered, but the problem involves setting up and evaluating a triple integral which is covered.  I will take the percentage of the points earned for these problems, and average this score with your original Exam 3 score to obtain a modified Exam 3 score.
  • The modified exam 3 score for the purposes of determining your final grade.
Caution: The alternative final exam policy puts more weight on the final exam.  Unless your Exam 3 score was low and you are confident that you will do substantially better with this material on the final, I do not recommend taking the risk.



Maple problems.

All maple problem sets are written as a Maple worksheet, so you must use maple to open them up and use them.

Downloading instructions: I will assume that you are already logged into a machine such as a composer terminal that has maple on it.  There are two ways to procede.

  1. Use the File -> Open URL option in maple to directly open the file in Maple.  I've listed the URL's below so that you can cut and paste them into Maple.
  2. Download the file and open it on your machine.  This task may vary somewhat depending upon which browser you use.  Move your mouse cursor to the link that  you wish to download, and then press and hold the right button.  Select the option "Save Link Target As".  Release the button and a Save File dialog box will open.  Click OK and the file will be downloaded into your account.  Now, you can use maple to open the file.  To do this, you can either enter "xmaple <filename>" from a command prompt (assuming you are using a sane operating system that has command prompts) or you can fire up Maple and then read in the worksheet using File -> Open.
