Course materials.

Some quick comments on file formats.

Generally, HTML looks the worst when printed but can be viewed by a browser.  PDF is compact way to print or view files.  The acrobat viewer is slow, but free.  Printer output is readable, but not great.

Matlab items.

  • Functions to generate regions of absolute stability.

    • Midpoint rule.
    Second order Adams-Bashforth.
    Fourth order Adams-Bashforth.
    Fourth order Adams-Moulton.
    Fourth order Adams-Bashforth-Moulton.
    A diary using these to generate the regions of absolute stability.
  • Functions for the "method of lines" demonstrations.

    • Explicit method of lines.
    Implicit method of lines.
    A diary using the explicit method lines routine.
  • Function for demonstrating first order upwinding of the linear wave equation.

    • Upwinding method.
    Centered differences (unstable).
    A diary of what we did in class.
  • Function for demonstrating the Lax-Wendroff method for solving the linear wave equation.

    • Lax-Wendroff method.
    Diaries of the in-class demonstrations with discontinuous and continuous initial data.
  • Functions for demonstrating ADI methods with various boundary conditions.
    adi1.m - ADI with homogeneous Dirichlet conditions.
    adi2.m - ADI with three homogeneous Dirichlet conditions and one homogeneous Neumann condition in the x direction.
    adi3.m - ADI with two homogeneous Dirichlet conditions and two homogeneous Neumann conditions, one in the x direction and one in the y direction.
    adi4.m - ADI with two Dirichlet conditions, one homogeneous and one inhomogeneous, and two homogeneous Neumann conditions, one in the x direction and one in the y direction.
    Here is a demo script for executing any one of these.
