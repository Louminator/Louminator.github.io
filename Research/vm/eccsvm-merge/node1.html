Introduction

  Vortex methods approximate fluid flows by expressing the vorticity field of a fluid as a linear combination of moving basis functions. Each basis function moves with the velocity field of the fluid evaluated at its center. The velocity field is recovered from the basis functions using a Biot-Savart integral. Though some investigators use single point vortices for flow calculations, most use localized distributions of vorticity known as ``vortex blobs.'' Viscosity can be incorporated a number of ways including random walks, circulation redistribution and core spreading [2, 8, 10, 9, 3, 12, 15]. The latter technique captures viscous diffusion by allowing the individual blobs to spread at a rate related to the fluid viscosity. The core spreading methods are deterministic, but also requires an additional internal degree of free in the computational element.

While core spreading vortex methods facilitate effective, accurate and naturally adaptive simulations of fluid flows, they suffer the disadvantage of relying up adaptive refinement wherein a single wide computation element must be replaced by two or more thinner elements to retain spatial accuracy. One possible remedy for this problem is to merge nearly overlapping elements. This strategy is substantiated both physically and mathematically: Two or more nearly-overlapping, same-signed computational elements in vortex computations can be thought to have extraneous degrees of freedom. Mathematically, replacing them with a single element induces a controllably small change in the approximate vorticity field. Physically, the a collection of nearly-overlapping elements have nearly the same dynamics as a single element because they experience nearly the same velocity field.

The basic tenets of the merging approach for spreading Gaussian elements can be extended to deforming elliptical Gaussian elements though with considerable complications. Since deforming elliptical Gaussians possess more internal degrees of freedom, the definition of ``nearly-overlapping'' must be extended mathematically. This extension can be summarized as follows:

  • The error induced by merging is defined to be the difference between a group of elements and the single element with which they are replaced. This error can be decomposed into contributions from differences in position, orientation, width and aspect ratio.
  • Both proximity and orientation are crucial in merging elements with a small induced error.
  • Proximity must be defined relative to the major and minor axes of the elliptical Gaussian elements as well as their widths.
  • Elements whose major axes are misaligned may still be merged if their aspect ratios are sufficiently close to unity (i.e. if the elements are nearly circular.)
  • All of these principles, when reduced to the axisymmetric case, recover and improve the results in [13].

This manuscript is organized in the following manner. In §2, we review the vortex method based on deforming elliptical blobs and introduce the necessary notation to formulate the merging criteria later in the paper. A more detailed description of the vortex method, its dynamics and its implementation can be found in [14]. In §3, we derive several theorems that bound the error induced by replacing nearly overlapping elements with a single element. In effect, these theorems define the meaning of ``nearly overlapping.'' In §4, these theorems are used to develop a localized merging algorithm for merging simulations. Finally, §5 walks through a detailed example using this algorithm.
